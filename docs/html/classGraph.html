Detailed Description

Weighted graph with pointer-based adjacency lists and Dijkstra routing.

Vertices are stored in a singly-linked list (vertices_head_ -> vertices_tail_). Each vertex owns a singly-linked list of outgoing Edge objects. addEdge() can create bidirectional (undirected) edges automatically.

Copy construction and copy assignment are disabled; move semantics are provided so the graph can be transferred without copying the entire vertex/edge structure.
