Detailed Description

Command-line entry point for the Fire Evacuation simulator.

Loads a building scene from a JSON file in the data/ directory, prints the initial evacuation plans for every room, then runs the time-step simulation until it finishes. The scene is read from disk so the project satisfies the "read input from data/" requirement.

Usage: ./build/sim_demo [input_file] If no file is given, data/input_default.json is used.
