Detailed Description

template<typename T>
class LinkedList< T >

A generic singly-linked list backed by pointer-based Node objects.

Provides O(1) push/pop at both ends and O(n) search/remove. Memory is managed internally: all nodes are heap-allocated and freed by the destructor or clear().

Template Parameters
TThe element type. Must be copy-constructible.
