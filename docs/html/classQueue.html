Detailed Description

template<typename T>
class Queue< T >

A generic first-in, first-out (FIFO) queue.

Backed by a pointer-based LinkedList. Enqueue is O(1) (push_back), dequeue is O(1) (pop_front). The queue owns the memory of its elements through the underlying list.

Template Parameters
TThe element type. Must be copy-constructible.
